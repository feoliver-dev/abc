O que é e para que serve o HTML

HyperText Markup Language, ou Linguagem de marcação de hipertexto, é utilizada para montar estruturas de páginas web.

O HTML cria os materiais utilizados para página, mas para apresentar e melhorar a expêriencia do usuário, é necessário integrar com o CSS e o JavaScript.

É uma linguagem baseada em tags(etiquetas), que indicam o conteúdo dentro delas. Comunicam ao navegador como esse conteúdo será exibido. São importantes para leitura acessivel e para especificar detalhes do site para os navegadores.

Elementos que da Head

Na HEAD fica o conteúdo que não é disponibilizado ao usuário, mas útil para que o navegador saiba as especificações do site. Única informação visivel ao usuario é o title que aparece na aba do navegador.

Dentro da tag META ficam duas informações importantes, o charset que informa como o conteúdo em html poderá ser representado pelo navegador na pagina do site.

O viewport é outra informação importante dentro da tag META, que informa como aquele site será visualizado em dispositivos mobiles, como celulares e tablets.
